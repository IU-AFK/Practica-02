## Seminario Python

Este repositorio contiene algunos de los ejercicios de la segunda actividad de la materia **Seminario de Python 2026**

La estructura del proyecto está dada por la consigna

Los notebooks con la resolución de los ejercicios se encuentran en la carpeta \notebooks
Y las funciones utilizadas para resolver los ejercicios se encuentran en la carpeta \src

## Estructura del proyecto

python-jupyter-notebook
|-- notebooks
    |-- ...
|-- src
    |-- utils.py
|-- README.txt
|-- .gitignore

## Requisitos previos
** Python 3.12.9 **

## Organización

Los notebooks deben ejecutarse desde la carpeta \notebooks

El código con las funciones necesarias para la ejecución deben ubicarse en \src

> Nota: El codigo no fue probado en Visual Studio
